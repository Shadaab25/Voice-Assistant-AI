# Voice-Assistant-AI

A Python-based **Voice Assistant** that listens to your commands and performs tasks like opening websites, playing videos, telling jokes, checking the time, and more — all through your voice.

Features

**Open Websites**
- LinkedIn  
- GitHub  
- Gmail  
- YouTube  
- Netflix  
- Amazon Prime Video  
- Google Search  
- Chrome Browser  

**Fun & Utility**
- Tells the current time  
- Plays songs or videos on YouTube  
- Tells random jokes  
- Reads short Wikipedia summaries  
- Opens weather updates  
- Can be stopped with “Stop”, “Exit”, or “Quit”  

---

## Technologies Used
- **Python 3.13**
- `SpeechRecognition`
- `pyttsx3`
- `pywhatkit`
- `webbrowser`
- `wikipedia`
- `pyjokes`
- `datetime`
- `pyaudio`
